# Tokenised Digital Magazine Marketplace

## Project Title
Tokenised Digital Magazine Marketplace

## Project Description
A decentralised marketplace where readers pay using tokens to access digital magazines. Publishers can upload magazine listings, and users can purchase them through blockchain transactions.

---

## Project Vision
To create a transparent, censorship-resistant, and token-driven magazine ecosystem where publishers earn fairly and readers gain seamless access without intermediaries.
